## Cloudflare Pages email setup
Set these environment variables in your Pages project:
- RESEND_API_KEY
- CONTACT_TO_EMAIL
- CONTACT_FROM_EMAIL (optional)

Recommended:
- CONTACT_TO_EMAIL = shahabasiddique@gmail.com
