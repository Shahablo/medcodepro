The design of the app is based on a simple, single-user interface that allows the user to enter and compare job offers. The app starts with a main menu that allows the user to enter or edit their current job details, enter job offers, adjust the comparison settings, or compare job offers.

When choosing to enter current job details, the user is presented with a user interface to enter or edit all the details of their current job. The user can then save the job details or cancel and exit without saving, returning in both cases to the main menu.

 
When choosing to enter job offers, the user is presented with a user interface to enter all the details of the offer. The user can then save the job offer details or cancel. The user can also enter another offer, return to the main menu, or compare the offer with the current job details.

When adjusting the comparison settings, the user can assign integer weights to the various factors, such as yearly salary, retirement benefits, etc. If no weights are assigned, all factors are considered equal.

When choosing to compare job offers, the user is presented with a list of job offers, displayed as Title and Company, ranked from best to worst, and including the current job (if present), clearly indicated. The user can then select two jobs to compare and see a table comparing the two jobs. The user is then offered to perform another comparison or go back to the main menu.
 

The ranking of jobs is based on a score that is computed as the weighted sum of the various factors, such as yearly salary, retirement benefits, etc. The weights can be adjusted in the comparison settings.
